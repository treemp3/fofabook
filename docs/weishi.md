# 大乘无我观

本书从悟前与悟后来宣说大乘佛法之无我观，解说二乘所证解脱道与大乘佛菩提道之异同；并宣示证悟后进修佛地之次第道，宣说诸地所应亲证之现观境界，如实显示成佛之道。
