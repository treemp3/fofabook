# 大乘无我观

本书从悟前与悟后来宣说大乘佛法之无我观，解说二乘所证解脱道与大乘佛菩提道之异同；并宣示证悟后进修佛地之次第道，宣说诸地所应亲证之现观境界，如实显示成佛之道。

## 辨唯识性相

本书对诽谤三乘菩提根本心阿赖耶识之邪见，加以教证上及理证上之分析与辨正，可免学人再被藏密应成派中观邪见所误导，可免再被假名施设而无实质之“幻想真如”者邪见所误导，熟读之后，未来可以渐具大乘见道之因缘。
