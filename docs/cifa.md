# 起信论讲记

详解大乘起信论心生灭门与心真如门之真实意旨，消除以往大师与学人对起信论所说心生灭门之误解，由是而得了知真心如来藏之非常非断中道正理；亦因此一讲解，令此论以往隐晦而被误解之真实义，得以如实显示，令大乘佛菩提道之正理得以显扬光大；初机学者亦可借此正论所显示之法义，对大乘法理生起正信，从此得以真发菩提心，真入大乘法中修学，世世常修菩萨正行。平实导师演述，共六辑，每辑三百余页，六辑都已出版。
